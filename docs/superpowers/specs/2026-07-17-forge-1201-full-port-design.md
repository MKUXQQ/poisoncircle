# Forge 1.20.1 毒圈完整移植设计

## 目标

将 `forge-1.20.1` 升级为与 NeoForge 1.21.1 版本一致的五阶段大逃杀毒圈：世界空间边界、同步的缩圈状态、安全区探测器和 Xaero 地图圆环。

## 运行模型

服务端拥有每个维度唯一的毒圈状态。第一阶段使用 `/poisoncircle center` 已设置的中心，之后阶段在当前安全区内选择满足世界边界与平地检查的下一圆心。等待期间圆心和半径不变；缩圈期间二者按同一进度线性插值。

服务端每次状态变化和每个游戏刻向客户端同步当前圆、下一圆、阶段、等待/缩圈剩余时间以及已公开的预览。第五次缩圈结束后全世界进入毒圈；所有玩家死亡后自动停止。

## 可见效果

客户端根据同步状态，在世界坐标中绘制红色半透明原版屏障风格圆柱墙。几何体以圆心与半径生成，绝不使用相机坐标，因此玩家靠近墙体时墙体不会贴到视角前方，也不阻挡移动。

## 安全区探测器

注册名为“安全区探测器”的指南针模型物品。管理员可使用命令获得；玩家右键会消耗一件（创造模式不消耗），只对该玩家揭示下下圈。地图界面和 Xaero 覆盖层显示玩家黄色、原版队友绿色；不显示其他玩家。当前圈为红色，下一圈白色，私有下下圈为青蓝色；公开后由白色替代。

## Xaero 兼容

以 `compileOnly` 方式使用本地 Xaero 小地图与世界地图 Jar，不把 Xaero 类打进毒圈 Jar。客户端在检测到两个 Mod 后注册各自覆盖层；覆盖层仅绘制圆形边界，不绘制当前或下一圈中心点。每帧读取同步后的插值状态，所以地图移动速度与右上角倒计时一致。

## 命令与规则

保留 `/poisoncircle start [初始半径] [最终半径]`、`center`、`time <圈数> <等待秒数> <缩圈秒数>`、`damage <基础> [每圈增加]`、`status`、`stop`、探测器获取和阶段命令执行配置。指令权限等级为 2。圈外伤害直接扣血并触发真实死亡、附加中毒、屏幕红闪和轻微抖动；创造与旁观模式免疫。

## 验收

1. `/poisoncircle start 100 1` 后可从世界任意位置看到独立红色圆柱边界。
2. 等待时世界与地图圆环不移动；缩圈时两者与右上角秒数同步平滑移动。
3. Xaero 小地图、世界地图分别显示红/白/青蓝圆环，无中心图标。
4. 安全区探测器只向使用者显示下下圈，且正确消耗。
5. Forge 1.20.1 `clean test build` 通过，并产出 `v1.1.0` Jar。
